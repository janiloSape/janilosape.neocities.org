★ free resources


i am not affiliated with any of these websites, i just like them and find them useful :D

these all:

    -have primarily free tools

    -use little to no ai (its getting more and more difficult to weed out the ai, but i tried my best)

    -are open source (i lied only some of them are open source sorry guys)


academic - scientific - literature


physics classroom - online tutorials for high school physics and chemistry

worldcat - a tool for finding items available at local libraries

poetry foundation - lots of poetry-related resources

base - a search engine for online academic resources and literature

arXiv - an archive of scholarly articles relating to physics, mathematics, etc.

trilium notes - a really cool open source software for taking notes

science news - a nonprofit that provides general interest science journalism

popular mechanics - also general interest science journalism

internet archive - a library of basically everything

anarchist library - an online archive of tons of texts relating to anarchism

openstax - educational resources and textbooks

khan academy - a Lot of learning resources and courses in multiple subjects

project gutenberg - an online library with free books in the public domain

einstein.caltech.edu - an archive of albert ensteins papers, letters, etc.


art references - drawing software - original characters


krita - a free, open-source platform for digital art creation

line of action - reference photos and practice tools for artists

opentoonz - a free, open-source digital 2d animation program

blender - a free, open-source program for 3d, 2d, and mixed-media art and animation

toyhouse - a website for storing oc artwork, literature, and information

^^an invite code is necessary to sign up. i currently have about 100 built up, if you need one hmu or something

likes and dislikes generator - a generator for character likes and dislikes


coding - programming - computers


web accessibility quick guide - a short guide on making your website more accessible

filegarden - a (usually) reliable free file hosting website

w3schools - free tutorials on coding in html, css, javascript, python, sql, etc.

ubuntu - a free and open-source os

github - a website for storing and showcasing original code

codeberg - github but better and cooler honestly

petrapixel layout generator - a very useful tool for creating a layout on neocities


toki pona


lipu sona pona - a good course for learning toki pona: en, eo, es, de, pl, pt, ru, zh

wasona - another good course: en, cs, de, es, fr, hu, pl, pt, ru, uk, ar, he

lipu sona mi pi toki pona - another good course: en, fr, it, jp, no

naming yourself in toki pona - a guide on translating your name to toki pona

duostories - duolingo stories in toki pona

toki pona lessons - toki pona video lessons by jan Misali

beginner texts in toki pona - toki pona stories for beginners


miscellaneous


cobalt - a tool for downloading media from websites (youtube is no longer supported)

jdownloader - another tool for downloading media from websites (youtube is supported)

allsides - a website that rates the bias of various news organizations and articles

ifixit - repair guides for a wide variety of things

mii creator - an online mii creator with many features and different rendering options
